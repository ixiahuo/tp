# Big Brother
[![CI Status](https://github.com/AY2526S2-CS2103T-T09-1/tp/actions/workflows/gradle.yml/badge.svg)](https://github.com/AY2526S2-CS2103T-T09-1/tp/actions)
[![codecov](https://codecov.io/gh/AY2526S2-CS2103T-T09-1/tp/graph/badge.svg)](https://codecov.io/gh/AY2526S2-CS2103T-T09-1/tp/graph/badge.svg)

![Ui](images/Ui.png)

**Big Brother is a desktop application for managing your contact details.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using Big Brother, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing Big Brother, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
